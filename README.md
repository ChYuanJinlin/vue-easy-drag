# vue-easy-drag
vue3简单的拖动

DirectiveBinding <{ isMove?: boolean, target?: string }> 指令参数

| 属性   | 描述                                             | 默认值         |
| ------ | ------------------------------------------------ | -------------- |
| isMove | 是否需要拖动                                     | true           |
| target | 当前需要拖动的目标,需要传需要拖动元素的class过来 | 当前绑定的元素 |
